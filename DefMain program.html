In the  Learning Resources text and videos, the Python commands shown are being executed in a Python shell.

But this is unique to Python and maybe a few other IDE tools. As our programs get larger and more complex, it will become more confusing to follow the flow of your program.

A more conventional approach that is consistent with most other languages is to Define a Main program. This is done with the def main(): command and all other code is indented (the same) afterwards. Using this approach will help you keep track of the flow of your program as all code will be inside the main or inside a function(later on). By convention everyone knows that the code starts with the main program.

Example

def main():

     print("Hello class")

     print("My name is Joe")

     print("My class is CMIS 102/6677")

#However, to the compiler, the above just defines the main program.

#To execute the program, you need ...

main()
