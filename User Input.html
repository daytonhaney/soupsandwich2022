Sometimes we would like to take the value for a variable from the user via their keyboard. Python provides a built-in function called input that gets input from the keyboard.

When this function is called, the program stops and waits for the user to type something. When the user presses Return or Enter, the program resumes and input returns what the user typed as a string.  Note: the default type from the input command is a string.

Let’s pair an input with a print to see how user input works:

user_typed = input()
print("user typed", user_typed)

Before getting input from the user, it is a good idea to print a prompt telling the user what to input.

print("What day is today? ")
day = input()

Python also allows you to combine the prompt with the input command.  You can put the prompt inside the input function, with quotes of course because the prompt is a string.

day = input("What day is today? ")
print("Today is", day)

If we prefer to have the user's input to be entered on the next line, we can include a special character sequence at the end of our prompt. The character sequence \n represents a newline, which is a special character that causes a line break. Character sequences that begin with \ are often referred to as escape characters and are used to represent characters that do not have an associated glyph. Let’s modify our previous program to include \n at the end of the prompt. Note: the \n is inside the quotes because is is a character. Also try the following with a \t which is a tab.

day = input("What day is today?\n")
print("Today is", day)

Note in the above examples, the user inputs have been strings. Also, remember that the default type from the input command is a string. But what if we expect the user to enter a number?  There is another function called the eval function. It will convert the return from the input command to a number.

If you expect the user to type an integer, you can try to convert the return value from the input command to int using the eval function.

prompt = "What is your favorite number? "
favorite = eval(input(prompt))
print("My favorite number is", favorite + 1, "one more.")
